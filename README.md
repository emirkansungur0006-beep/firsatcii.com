# Fırsatçı - İhale ve İş Platformu

Bu proje bir Next.js frontend ve NestJS backend içeren monorepo yapısıdır.

## Yapı
- `apps/web`: Next.js Frontend
- `apps/api`: NestJS Backend
- `packages/shared`: Ortak tipler ve yardımcılar

## Canlı Yayın
- Frontend: Vercel
- Backend: Render/Koyeb
- Database: Supabase
